Qwen Translator App

Esta aplicación traduce frases de inglés a español usando Qwen2-0.5B de manera local, sin API keys.

Ejemplos requeridos

  • I like soccer → Me gusta el fútbol
  • How are you? → ¿Cómo estás?
  • What time is it? → ¿Qué hora es?

Cómo correr la app

pip install -r requirements.txt
python app.py
